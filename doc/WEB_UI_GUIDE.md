# Web界面使用指南

## 概述

股票技术指标分析系统提供了优雅美观的Web界面，用于可视化展示技术指标、买卖信号和回测结果。

## 访问方式

1. 启动服务后，在浏览器中访问：
   ```
   http://localhost:5000
   ```
   或
   ```
   http://localhost:5000/index
   ```

## 功能说明

### 1. 参数设置

- **周期类型**：选择日线、周线或月线
- **计算指标**：点击按钮计算技术指标和买卖信号
- **回测分析**：点击按钮进行回测计算

### 2. 统计卡片

显示关键统计信息：
- 总记录数
- 买入信号次数
- 卖出信号次数
- 超卖区次数
- 超买区次数

### 3. 回测结果

显示回测统计：
- 总收益和总收益率
- 年收益率
- 交易次数
- 回测周期

### 4. 趋势线图表

使用Chart.js绘制的交互式图表，展示：
- 收盘价走势
- 趋势线
- 支撑位
- 阻力位

### 5. 数据表格

三个标签页：
- **买卖信号**：显示买入和卖出信号列表
- **最近数据**：显示最近20条关键指标数据
- **交易记录**：显示回测的交易记录（仅回测后显示）

## 设计特点

### 视觉设计

- **现代化UI**：采用卡片式布局，渐变背景
- **优雅配色**：使用专业的配色方案
- **响应式设计**：适配各种屏幕尺寸
- **流畅动画**：平滑的过渡效果和加载动画

### 交互体验

- **实时反馈**：加载状态提示
- **数据可视化**：图表展示趋势
- **清晰展示**：表格数据格式化显示
- **易于操作**：直观的按钮和表单

## 技术栈

- **前端框架**：原生HTML/CSS/JavaScript
- **图表库**：Chart.js 4.4.0
- **后端框架**：Flask
- **样式**：自定义CSS，采用现代设计规范

## 浏览器兼容性

支持所有现代浏览器：
- Chrome/Edge (推荐)
- Firefox
- Safari
- Opera

## 注意事项

1. 确保API服务已启动
2. 首次加载可能需要几秒钟
3. 图表数据按时间倒序显示（最新在前）
4. 回测功能需要输入有效的初始资金
